NexaCore, in more detail

NexaCore was built as a single deployable HTML file to keep hosting and handoff simple for a small operations team. It covers the day-to-day shape of the business: tracking which inverter and battery models are in stock, producing GST-compliant invoices at the point of sale, and scheduling follow-up notifications so customers don't have to be chased manually.

On the interface side, the build includes subtle 3D card tilt and parallax effects, scroll-reveal animation for longer pages, and canvas-rendered charts for the analytics dashboard — balanced against keeping the core workflows fast and distraction-free for daily use.
